# Análise de Sentimento - IMDb PT-BR 🎬

Este projeto utiliza técnicas de **Processamento de Linguagem Natural (NLP)** e **Machine Learning** para classificar avaliações de filmes em português como positivas ou negativas.

## 🚀 Objetivo
Desenvolver um pipeline completo de Ciência de Dados, desde o pré-processamento de texto até a avaliação de modelos de classificação, utilizando o dataset público do IMDb traduzido para o português.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.12
* **Bibliotecas:** Pandas, Scikit-Learn, NLTK, Matplotlib, Seaborn
* **Ambiente:** Linux (Ubuntu) / VS Code

## 📂 Estrutura do Projeto
* **data/**: Armazenamento do dataset (não incluído no controle de versão).
* **notebooks/**: Jupyter Notebooks com análise exploratória e experimentos.
* **src/**: Scripts Python com funções auxiliares de limpeza e processamento.
* **requirements.txt**: Lista de dependências para reprodução do ambiente.
